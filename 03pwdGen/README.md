# Password Generator (03pwdGen)

A small React demo covering `useState`, `useCallback`, `useEffect`, and `useRef`.

## Concepts demonstrated

- **useState** — length slider, checkbox toggles, generated password
- **useCallback** — memoized `passwordGenerator` and `copyPasswordToClipboard`
- **useEffect** — regenerate password when length or options change
- **useRef** — select input text before copying to clipboard

## Run locally

```bash
cd 03pwdGen
npm install
npm run dev
